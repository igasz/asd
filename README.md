### Algorithms & Data Structures | Algorytmy i Struktury Danych

Repozytorium zawierające moje implementacje popularnych algorytmów i struktur danych w języku Python. 
A collection of my implementations of popular algorithms and data structures in Python.

## Project Structure / Struktura Projektu

### 1. Sorting & Basics / Sortowania i Podstawy
Podstawowe algorytmy sortowania oraz operacje na strukturach.
* `sortowania/` (`bubble_sort.py`, `bucket_sort.py`, `count_sort.py`, `heap_sort.py`, `insertion_sort`, `merge_sort.py`, `quick_sort.py`, `radix_sort.py`, `selection_sort.py`, `sorting_linkedlist.py`)
* `binary_search.py`
* `insert_delete_heap.py` 
* `quickselect.py` 

### 2. Graph Algorithms / Algorytmy Grafowe
Implementacje najważniejszych algorytmów pracujących na grafach.
* `articulation points.py` 
* `bellman-ford.py` 
* `bfs.py` / `dfs.py` 
* `bridges.py` 
* `cykl Eulera.py`
* `dijkstra.py` 
* `find_union.py`
* `floyd-warshall.py`
* `kruskal.py`
* `topological sort.py`

### 3. DP & Greedy Algorithms / Programowanie Dynamiczne i Zachłanne
Algorytmy optymalizacyjne i rozwiązywanie złożonych problemów.
* `kadane.py` 
* `lis.py` (Longest Increasing Subsequence)
* `problem plecakowy.py` - Knapsack Problem
* `travelling salesman problem.py`
